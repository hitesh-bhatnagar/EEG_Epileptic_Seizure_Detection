AI/ML Project: CNN Binary Classification Model

Welcome to the repository for the CNN Binary Classification model project! This project utilizes Convolutional Neural Networks (CNNs) for binary classification tasks.

Cloning the Repository

To get started, clone this repository to your local machine using Git Bash or your preferred terminal:

git clone https://github.com/hitesh-bhatnagar/EEG_Epileptic_Seizure_Detection.git
cd EEG_Epileptic_Seizure_Detection

Branches

This repository contains multiple branches, each serving different purposes. Below are the branches you can reference:

main Branch

The main branch contains the stable version of the project. It is the default branch where you can find the latest tested and reviewed changes.

git checkout main

master Branch (CNN Binary Classification Model)

The master branch specifically hosts the CNN Binary Classification model. You can access and use this branch to explore or integrate the CNN model into your projects.

git checkout master

CNN Binary Classification Model

The CNN Binary Classification model in the master branch achieves a validation accuracy of 55.16%. This model is trained to classify data into two distinct classes using convolutional layers.

Usage

You can integrate the CNN model into your own projects by cloning the repository and switching to the master branch as shown above. Refer to the model implementation and adapt it to your specific use case.
